About grids

A grid system is a key function of any CSS framework. It's a way of arranging elements on a page to make them simple, neat and responsive.

A grid conisist of rows and columns. In a typical framework (and in Magic), a row contains 12 spaces for columns. Within a row, you have columns, which each occupy a specified number of spaces. In one row, the number of spaces occupied therefore has to add up to 12. In the below example, there are 12 columns, each occupying 1 space. 12 x 1 = 12 so the columns add up to complete the row.

The number of column spaces is almost always 12 across all frameworks, because 12 has many convenient factors. It allows regular layouts of 12@1, 4@3, 3@4, 2@6 and 1@12.

Another key feature of columns is that they pile on either mobile or desktop. Also, each column is reset to 100% width to appear neatly. Try resizing the screen to see what happens to the rows above.

Magic grids

Grids in magic are as simple as they could possibly get, and require minimal markup. To make a row (which contains columns), simply create a <div class="row">:

<div class="row">
	...
</div>

Inside a row, you need to place columns. A column is defined by using <div class="col colX">, where X is a number between 1 and 12:

<div class="row">
	<div class="col col6">
		<div>
			<p>Hi!</p>
		</div>
	</div>
	<div class="col col6">
		<div>
			<p>Hi!</p>
		</div>
	</div>
</div>

No gutter

To remove the gutter (the spacing) between columns, simply omit the <div> from within the <div class="col colX">. For example:

<div class="col col6">
	<p>Hi!</p>
</div>

Piling on tablet

By default, rows will pile on mobile (425px) and below. However, it's sometimes useful to change this to tablet (768px). To do this, just add the class pile-tablet to the row div.
