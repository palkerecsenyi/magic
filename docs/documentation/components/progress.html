Progress New!

A very simple progress bar component with a customisable colour, width and height.

25%
50%
75%
100%

Basic layout

The basic layout of a progress bar is as follows:

<div class="progress">
  <div class="bar SIZE COLOUR" style="width:WIDTH%">TEXT</div>
</div>

In this case, SIZE refers to the height of the progress bar. You can choose from:

  • small
  • regular
  • large
  • xlarge

COLOUR refers to any Magic colour (including darkens and lightens). WIDTH refers to a percentage which defines the current 'progress' of the bar. TEXT can be any textual content to show in the middle of the progress bar - this must not be empty, as the height of the progress bar will not work properly otherwise.

Functionality

Magic.css does not include any JavaScript, and you must add your own in order to add functionality to Magic progress bars. All you need to do is increase the width CSS property of the <div class="bar">, up to a maximum of 100%. A width transition is applied automatically, so the width will be animated if a change in percentage occurs.

Here is an example:

0%
<div class="progress">
  <div class="bar regular blue" style="width:0%" id="bar">0%</div>
</div>
<button class="btn btn-blue" onclick="progress()">Start</button>
<script>
  function progress(){
    var obj = document.getElementById("bar");
    var interval = undefined;
    if(interval===undefined){
      interval = window.setInterval(function(){
        var width = obj.style.width;
        if(width!=="100%"){
          width = width.split("%");
          width[0] = parseInt(width[0]);
          width[0]+= 5;
          width = width.join("%");
          obj.style.width = width;
          obj.innerHTML = width;
        }else{
          window.clearInterval(interval);
        }
      },500);
    }
  }
</script>

You can also change the colour of the progress bar using JS, and the transition will be applied automatically:

0%
<div class="progress">
  <div class="bar regular blue-lighten" style="width:0%" id="bar2">0%</div>
</div>
<button class="btn btn-blue" onclick="progress2()">Start</button>
<script>
  function progress2(){
    var obj = document.getElementById("bar2");
    var interval = undefined;
    if(interval===undefined){
      interval = window.setInterval(function(){
        var width = obj.style.width;
        if(width!=="100%"){
          width = width.split("%");
          width[0] = parseInt(width[0]);
          width[0]+= 5;
          width = width.join("%");
          obj.style.width = width;
          obj.innerHTML = width;
        }else{
          window.clearInterval(interval);
          obj.classList.remove("blue-lighten");
          obj.classList.add("yellow-lighten");
        }
      },500);
    }
  }
</script>
