Basic progress bar

Magic uses the HTML5 <progress> tag to create styled progress bars. As a result of this, the required syntax is extremely simple. To create a progress bar, you need to choose a colour and a size. The template for a Magic progress bar is as follows:

<progress class="progress progress-COLOUR (progress-SIZE)"></progress>

You can replace 'COLOUR' with any standard Magic colour. This means you can not use darkens or lightens. Refer to the colour docs for more info. Here are all supported colours, in both determinate and indeterminate forms (in their 'small' version to conserve space):

If you do not include the 'progress-SIZE' class, a default size will be applied. If you do include it, you can choose from the following sizes:

Class Height Border radius Demo
progress-small 1rem 10px
None 1.25rem 10px
progress-large 1.5rem 15px
progress-xlarge 1.75rem 20px

Determinate vs. indeterminate

If a progress bar does not have a value attribute, it will automatically appear full and it will have stripes. Otherwise, the value attribute (along with the max attribute) defines how 'full' the progress bar should be. Here are two examples:

<progress value="50" max="100" class="progress progress-blue"></progress>
<progress class="progress progress-blue"></progress>
