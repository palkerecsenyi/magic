Structure

Magic pagination components are quite simple to build. They use flexbox to position elements in one line, and are responsive on tablet.

They consist of 3 root level sections that should be contained in a <div class="pagination"> and can be placed in any order:

Section Class name Description
Left button pagination-left This should be a button, typically with the text 'Previous'. You can create this button using the normal ,btn class system (see the button docs).
Right button pagination-right Same as above, but is placed on the right and usually contains the text 'Next'.
Pagination list pagination-list Must be a <ul class="pagination-list">. This can also contain an additional pagination-COLOUR, where colour is a standard Magic colour - this will set the colour that active items should be. The list should contain (multiple) <li class="pagination-item">s, which can each either contain a <a class="pagination-link" href="#"> or have an additonal pagination-skip class and contain &hellip;. pagination-link tags can also have the active class, which will give them the background colour defined previously and therefore make them appear active.
